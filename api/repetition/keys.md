|     NAME     |  TYPE   |                          DESCRIPTION                           |
|--------------|---------|----------------------------------------------------------------|
| `ignorecase` | `bool`  | Makes all matches case-insensitive.                            |
| `alpha`      | `bool`  | Limits all matches to alphanumeric tokens.                     |
| `tokens`     | `array` | A list of tokens to be transformed into a non-capturing group. |
