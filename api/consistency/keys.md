|     NAME     |  TYPE  |                            DESCRIPTION                             |
|--------------|--------|--------------------------------------------------------------------|
| `nonword`    | `bool` | Removes the default word boundaries (`\b`).                        |
| `ignorecase` | `bool` | Makes all matches case-insensitive.                                |
| `either`     | `map`  | A map of `option 1: option 2` pairs, of which only one may appear. |
