Introducing Vale Server

Your style, our editor

Vale Server is a cross-platform ( and ) desktop application that brings your editorial style guide to life. Get Started Documentation
  • Native Performance

    Vale Server is a native desktop application. There are no size restraints, usage quotas, or other limitations.

  • Extensible

    Vale Server provides a simple, well-documented API for integrating with third-party applications.

  • Private & Secure

    Vale Server runs locally, ensuring that you never have to send your content to a remote server.

NOT ANOTHER GRAMMAR CHECKER

Control your content

Vale Server doesn’t attempt to teach you how to write; it’s a tool for writers. It focuses on writing style—allowing you to control your voice, terminology, branding, and more.
  • Choose your style

    Styles are folders consisting of simple, easy-to-share YAML files. You can create your own or use one from our existing collection.

    Learn more ›
  • Start a project

    Manage and switch between different configurations for each of your writing projects.

    Learn more ›
  • Use your Vale configuration

    Vale Server offers complete compatibility with the Vale CLI, including a Compatibility Mode that makes use of local configurations.

Intuitive web-based user interface

Use the browser, not the command line

Vale Server comes with a full-featured, web-based dashboard for managing your content.
  • Multiple Vocabularies

    Create and manage multiple vocabularies for each of your projects.

  • A managed StylesPath

    Find, install, and update styles from our community-maintained library.

  • Create & test rules

    Create and run new rules directly from your browser, including the ability to view the compiled rule in regex101.
